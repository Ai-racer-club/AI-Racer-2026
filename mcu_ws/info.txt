okay so awesome it appears we got an environment for the microcontrollers
but.....we need to allow the host machine give permissions to the container 
and we can do so via the usbipd applet!!
go to your windows powershell(run as admin for easy access!)
1.Install the tool needed to know the devices listed on your USB
	winget install --interactive --exact dorssel.usbipd-win
^^^^^^ equivalne of an apt install
2.List the avaliable options before you
	usbipd list
3.Bind your usb device to your windows subsystem for linux
	usbipd bind --busid <your-BUSID> <---look for it! its in there!
4.Force windows to give the specific usb device to your docker environment
	usbipd attach --wsl --busid <your-BUSID>

then just to double check you can simply run the 
	--pio device list 
in your container terminal to see if we have it!
